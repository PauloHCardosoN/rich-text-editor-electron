## Getting Started

### Installation

1. Clone the repo

```sh
git clone https://github.com/PauloHCardosoN/rich-text-editor-electron.git
```
2. Install NPM packages
```sh
npm install
# or
yarn install
```

3. Run the project:
```sh
npm run start
# or
yarn start
```
